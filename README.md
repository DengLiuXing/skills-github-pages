## ![star](icons/outline/star.svg) 关于这个博客

这是使用 **GitHub Pages** + **Jekyll** 搭建的个人博客站点。

- **无需服务器**：GitHub 免费托管，一键部署
- **Markdown 写作**：专注于内容，无需关心前端代码
- **版本控制**：每次提交自动构建发布

博客地址：[https://dengliuxing.github.io/skills-github-pages/](https://dengliuxing.github.io/skills-github-pages/)

---

## ![folder-open](icons/outline/folder-open.svg) 项目结构

```
skills-github-pages/
├── _posts/          # 博客文章（Markdown 格式）
├── _config.yml      # 站点配置文件
├── index.md         # 首页内容
└── README.md        # 项目说明
```

---

## ![rocket](icons/outline/rocket.svg) 快速开始

### 发布新文章

在 `_posts/` 目录下创建 Markdown 文件，命名格式：`YYYY-MM-DD-title.md`

```markdown
---
title: "文章标题"
date: 2026-07-15
---

文章内容...
```

### 本地预览

```bash
bundle install
bundle exec jekyll serve
```
